# 施工事例写真の追加方法

施工事例セクションは、この `works/` フォルダ内の案件フォルダを自動で読み込んで表示します。

## 基本ルール

案件ごとにフォルダを1つ作成してください。

```text
works/
  2026-001/
    before.jpg
    after.jpg
```

- `before.jpg` と `after.jpg` がある場合: 施工前・施工後として2枚表示
- `after.jpg` のみの場合: 施工後写真のみ表示
- `before.jpg` のみの場合: 施工前写真のみ表示
- `before.jpg` / `after.jpg` が無い場合: フォルダ内の画像をファイル名順に表示

対応画像形式は `jpg`, `jpeg`, `png`, `webp` です。

## 文字情報を入れたい場合

写真だけでも掲載できます。施工内容・工期・参考価格帯を表示したい場合は、同じフォルダ内に `info.json` を追加してください。

```json
{
  "content": "2台用カーポート新設",
  "period": "約1日",
  "price": "40万〜70万円台"
}
```

`info.json` が無い場合は、写真だけのカードとして表示されます。

## 公開方法

画像フォルダを追加して GitHub に反映すると、GitHub Pages 上の施工事例セクションに自動で表示されます。
